# Financial Analytics Platform
## AI-Powered Investment Decision System
### ACC102 Track 4 Coursework Project

### Project Overview
This is an interactive Streamlit financial analysis application completed for ACC102 Track 4 assessment.
The platform provides comprehensive multi-dimensional financial analysis for 5 listed home appliance manufacturing enterprises, including core financial metrics, stock trend analysis, investment portfolio evaluation, and market risk assessment. All interface content is fully in English, with complete interactive logic and data visualization functions.

### Supported Listed Companies
1. Gree Electric
2. Midea Group
3. Haier Smart Home
4. Supor
5. Joyoung

### Core Functionalities
- Multi-company independent data switching
- Custom date range filtering system
- 4 integrated analysis modules:
  1. Core Financial Metrics Analysis
  2. Historical Stock Price Trend Analysis
  3. Investment Portfolio Performance Analysis
  4. Financial Market Risk Assessment
- Multi-type professional data visualization (bar charts, line charts, scatter plots, correlation heatmap matrix)
- Standard financial indicator system: Revenue Growth, Net Profit Margin, Gross Margin, ROE, Beta Coefficient, VaR, Sharpe Ratio, Max Drawdown
- One-click CSV full data export function
- Standard color coding: green for positive growth, red for negative decline

### Target Audience
This tool is designed for accounting and finance undergraduate students, entry-level investors, and basic corporate financial analysis learners. It is built to demonstrate interactive financial data analysis logic in academic learning scenarios.

### Project Limitations
All internal financial and stock trend data is simulated for educational demonstration purposes. Real-time market data API access is not included in this version. Multi-company horizontal benchmark comparison is not implemented, the platform supports independent single-company data analysis only.
