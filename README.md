# Financial Analytics Platform
## AI-Powered Investment Decision System

### 1. Project Overview
This is a no-code interactive financial analysis dashboard completed for ACC102 Track 4 coursework.
The platform provides comprehensive financial data analysis for listed home appliance manufacturing companies. It supports independent data viewing of multiple enterprises, financial indicator query, stock trend visualization, risk assessment and data export functions.

### 2. Project Features
- Multi-company selection: supports switching among 10 listed enterprises
- Custom date range filtering
- Core financial metrics display (ROE, Gross Margin, Revenue Growth, Net Profit Margin)
- Multi-type data visualization: stock trend line chart, revenue bar chart, correlation matrix, risk-return scatter plot
- Investment portfolio & market risk assessment module
- One-click CSV data export function

### 3. Target Audience
This tool is designed for accounting & finance undergraduate students, entry-level investors, and learners of basic corporate financial analysis.

### 4. Data Description
All financial and stock data is used for academic learning purposes. The platform presents standardized financial indicators and simulated trend data to demonstrate interactive data analysis logic.

### 5. Limitations
The platform supports single-company independent data viewing. Multi-company horizontal comparative analysis is not included in this version. Future iterations can add peer benchmarking functions and real-time financial data API access.
