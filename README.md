# Home Appliance Giants Interactive Analysis Tool

## 1. Project Overview
This is an interactive data analysis tool built for ACC102 Track 4. It helps users analyze stock price trends and financial indicators of three leading Chinese home appliance companies.

## 2. Data Description
- Simulated stock price data from 2023 to 2025
- 2024 core financial indicators (ROE, gross margin, revenue growth)
- Companies: Midea Group, Gree Electric, Haier Smart Home

## 3. Features
- Interactive company selection and date filtering
- Financial indicator display
- Stock price trend visualization
- CSV data export
